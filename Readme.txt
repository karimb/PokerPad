PokerPad, originally developed by Xanderz
This version contains changes from the following 2+2 posters:
- Donkey111: Party Poker/Modern Style/Real Money Cash games)
- xor: original work on iPoker
On GitHub
- bl1 (Brother Love on 2p2): numerous fixes for iPoker
- karimb: Initial support for 888/pacific

The script requires AutoHotKey (NOT AutoHotKey_L) then run "PokerPad.ahk".
Or simply run "PokerPad.exe" if you do not wish to install AutoHotKey.
In order to work with iPoker, the chat window neeeds to be maximised.
